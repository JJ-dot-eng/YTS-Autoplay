# AGENTS.md

- 답변은 항상 한국어로 작성하고, 개발 지식이 적은 사용자도 이해할 수 있게 쉽게 설명한다.
- 이 저장소의 목적은 YouTube Shorts 영상이 한 번 끝나면 자동으로 다음 Shorts로 넘기는 브라우저 확장프로그램을 만드는 것이다.
- 구현은 최소 권한 원칙을 지킨다. 가능하면 YouTube Shorts 페이지에 필요한 권한과 코드만 추가한다.
- 기본 구조는 Manifest V3 Chrome 확장프로그램을 기준으로 한다.
- 자동 넘김 로직은 `content script` 중심으로 구현하고, YouTube DOM 변경에 대비해 안전하게 감지한다.
- 불필요한 외부 통신, 사용자 데이터 수집, 비밀 정보 접근은 하지 않는다.
- 설정 UI가 필요하면 간단한 popup 또는 storage 기반 옵션으로 추가한다.
- 변경 후에는 가능한 정적 검증, 확장 로드 확인, YouTube Shorts 수동 테스트 절차를 함께 점검한다.
- Git 저장소에서 검증이 통과하면 관련 변경만 로컬 커밋한다. 검증 실패나 의도치 않은 변경이 있으면 커밋하지 않고 이유를 보고한다.
