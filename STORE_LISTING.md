# Chrome Web Store 등록 메모

## 단일 목적

YouTube Shorts 영상이 끝나면 자동으로 다음 Shorts로 넘깁니다.

## 짧은 설명

YouTube Shorts를 한 편씩 자동으로 이어 볼 수 있게 해주는 간단한 확장프로그램입니다.

## 자세한 설명 초안

YTS Autoplay는 YouTube Shorts 영상이 끝났을 때 자동으로 다음 Shorts로 넘기는 Chrome 확장프로그램입니다.

YouTube Shorts 페이지에서만 실행되며, 현재 화면에 보이는 영상의 재생 종료를 감지합니다. 다음 버튼을 찾으면 버튼을 클릭하고, 찾지 못하면 아래로 스크롤해 다음 Shorts로 이동합니다.

개인정보를 수집하지 않고, 외부 서버로 데이터를 보내지 않으며, 별도 확장 권한도 요청하지 않습니다.

## 개인정보 항목

- 데이터 수집: 없음
- 외부 전송: 없음
- 광고/분석/추적: 없음
- 사용 권한: 없음

## 심사용 테스트 안내

1. 확장프로그램을 설치합니다.
2. `https://www.youtube.com/shorts/...` 형식의 YouTube Shorts 페이지를 엽니다.
3. Shorts 영상을 끝까지 재생합니다.
4. 영상 종료 후 다음 Shorts로 자동 이동하는지 확인합니다.
5. 일반 YouTube 영상 페이지에서는 동작하지 않는지 확인합니다.
